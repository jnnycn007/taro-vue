# taro-vue

*本跨端模板项目采用 `@tarojs/taro@^3.6.26` + `vue@^3.4.21`+ `typescript@^5.4.5` + `webpack@^5.91.0` + `@nutui/nutui-taro@^4.3.5` + `@nutui/icons-vue-taro@^0.0.9` 实现！*

## Project

```sh
git clone git@github.com:themusecatcher/taro-vue.git
```

## Install

```bash
cd taro-vue
pnpm install
```

## Run & Develop

```bash
cd taro-vue
pnpm dev:weapp
```
